# Code of Conduct

This project is committed to a welcoming, respectful, and harassment-free environment for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent while still being clear about risks and defects.
- Keep discussions focused on the project and its users.
- Accept feedback gracefully and help others contribute successfully.

## Unacceptable behavior

- Harassment, intimidation, discriminatory language, or personal attacks.
- Publishing private information without permission.
- Repeated disruptive behavior after maintainers ask for it to stop.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior harms the project community. If you need to report a concern, open a private security advisory or contact the maintainers through the repository owner profile.
